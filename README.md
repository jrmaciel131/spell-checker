## 🇺🇸 English

### What is it?
A Chrome extension that adds spell-checking and style correction to **any text field** on the web, using the free LanguageTool API.

### Features
- 🔴 **Red dot** when errors are detected — click to see suggestions
- 💡 **Purple dot** for style tips and improvements
- 〰️ **Wavy underline** directly in the text field
- ✨ **AI button** to rewrite with Gemini, Groq, Claude, GPT-4, or DeepSeek
- 📖 **Personal dictionary** of ignored words
- 🔇 **Silent mode** to pause everything
- ☀️ **Light/dark theme** configurable
- 🌐 **Interface in 6 languages**: EN-US, EN-GB, PT-BR, PT-PT, ES, FR

### Installation
1. Download the `.zip` file and extract the folder
2. Open `chrome://extensions` in Chrome
3. Enable **Developer mode** (top right corner)
4. Click **Load unpacked** and select the extracted folder

### Usage
- Click any text field — the indicator dot appears when you start typing
- Click the dot to view/fix errors
- Click ✨ to improve text with AI

### Keyboard shortcut
`Ctrl + Shift + Space` — Opens the AI suggestion on the active field

### Supported AI integrations
| AI | Cost |
|---|---|
| Gemini (Google) | Free |
| Groq (LLaMA) | Free |
| DeepSeek | Almost free |
| ChatGPT (OpenAI) | Paid |
| Claude (Anthropic) | Paid |

## 🇧🇷 Português Brasil

### O que é?
Extensão do Chrome que adiciona correção ortográfica e de estilo em **qualquer campo de texto** da web, usando a API gratuita do LanguageTool.

### Recursos
- 🔴 **Ponto vermelho** ao detectar erros — clique para ver sugestões
- 💡 **Ponto roxo** para dicas de estilo e melhorias
- 〰️ **Sublinhado ondulado** diretamente no campo de texto
- ✨ **Botão de IA** para reescrever com Gemini, Groq, Claude, GPT-4 ou DeepSeek
- 📖 **Dicionário pessoal** de palavras ignoradas
- 🔇 **Modo silencioso** para pausar tudo
- ☀️ **Tema claro/escuro** configurável
- 🌐 **Interface em 6 idiomas**: EN-US, EN-GB, PT-BR, PT-PT, ES, FR

### Instalação
1. Baixe o arquivo `.zip` e extraia a pasta
2. Abra `chrome://extensions` no Chrome
3. Ative o **Modo do desenvolvedor** (canto superior direito)
4. Clique em **Carregar sem compactação** e selecione a pasta extraída

### Uso
- Clique em qualquer campo de texto — o ponto indicador aparece ao digitar
- Clique no ponto para ver/corrigir os erros
- Clique em ✨ para melhorar o texto com IA

### Atalho de teclado
`Ctrl + Shift + Space` — Abre a sugestão de IA no campo ativo
